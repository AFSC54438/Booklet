**Booklet** is a Chrome extension that provides quick access to **Cambridge International (CIE) IGCSE and AS/A Level past papers** through a simple and clean interface.

It is designed for students who want fast, organized access to exam resources without navigating multiple websites.

---

## Download Instructions

### Download ZIP (Recommended)

1. Click the green **Code** button on this page  
2. Select **Download ZIP**  
3. Extract the ZIP file to a folder on your computer  
   **Do not delete this folder after installation**

---

## How to Load the Extension into Chrome

### 1. Open the Extensions Page
1. Open **Google Chrome**
2. Go to: `chrome://extensions/`
3. Press **Enter**

---

### 2. Enable Developer Mode
1. In the **top-right corner**, toggle **Developer mode** **ON**

---

### 3. Load the Extension
1. Click **Load unpacked**
2. Select the **extracted Booklet folder**  
   (the folder containing `manifest.json`)
3. Click **Select Folder**

---

### 4. Confirm Installation
- The **Booklet** extension should now appear in your extensions list
- (Optional) Pin it:
  - Click the **puzzle icon** (ðŸ§©) in the Chrome toolbar
  - Click the **pin** icon next to **Booklet**

---

## Important Notes
- **Do not move or delete the folder** after installation
- If Chrome restarts or the folder is removed, the extension will stop working

---

## Updating the Extension
1. Replace the old folder with the new version
2. Go to `chrome://extensions/`
3. Click **Reload** on the Booklet extension
